# ZOL
"Zig Of Life" - A terminal-based Conway's Game of Life implementation in Zig

## Controls

| Key | Action                  |
|-----|-------------------------|
| `q` | Quit                    |
| `p` | Pause/Resume            |
| `+` | Increase Speed          |
| `-` | Decrease Speed          |
| `n` | Next Step (when paused) |
